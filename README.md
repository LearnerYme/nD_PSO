# nD PSO

`code author: yghuang`

## Quick start

1. Change the fitness function as you want.

2. Change the hyperparameters in `main.cxx`.

3. `make` and `./run`.

# Python version

There is a python version (but only support 1d parameter space) in [this link](https://github.com/LearnerYme/1D_PSO).
